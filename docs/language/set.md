# DGQL/language/`SET`

## Intro

Setting values on either nodes or relationship properties

## Usage

### `SET` Value

```graphql
{
  SET {
    name(value: "Dan")
  }
}
```

### `SET` Date

See [DateTime TCK](https://github.com/danstarns/DGQL/blob/main/packages/language/tests/tck/tck-test-files/set/date.md)
